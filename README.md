# Data Strategy Portfolio

Portfolio pessoal de Sarah Zanardi, focado em Data Strategy, BI e Product Analytics.

## Stack

- HTML5
- CSS3
- GitHub Pages

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (exemplo: data-strategy-portfolio).
2. Envie os arquivos deste projeto para a branch principal:
	- index.html
	- styles.css
	- README.md
3. No repositório, acesse Settings > Pages.
4. Em Build and deployment:
	- Source: Deploy from a branch
	- Branch: main
	- Folder: / (root)
5. Salve e aguarde alguns segundos.
6. O site ficará disponível na URL informada pelo GitHub Pages.

## Personalizacao rapida

- Troque textos das secoes em index.html para refletir seus resultados reais.
- Atualize links de GitHub e LinkedIn no bloco de contato.
- Se quiser, adicione novas secoes de projetos com links para dashboards e notebooks.

## Estrutura

- index.html: estrutura e conteudo do portfolio
- styles.css: identidade visual e responsividade
- README.md: guia de publicacao
